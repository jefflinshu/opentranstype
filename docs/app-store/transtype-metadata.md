- Suggested App Store answer: uses only exempt system encryption provided by Apple platform frameworks.
- Rationale: the app is a local macOS utility that uses Accessibility APIs and Apple's on-device Translation framework. It does not implement custom cryptography, VPN, secure messaging, or its own encryption protocol.

## Review Notes

- This app is a macOS floating translation toolbar for text entry fields.
- The app requires Accessibility permission to read selected or active text from supported text inputs.
- Translation uses Apple's on-device Translation framework and installed Apple language packs.
- No login is required.
- No in-app purchase or subscription is included.
- No user-generated public content is included.
- If a text field does not expose standard Accessibility text attributes, the fallback flow is manual selection by the user.
